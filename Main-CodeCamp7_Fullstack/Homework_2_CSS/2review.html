Property Value Unit Differentiation
คุณสมบัติ ค่าที่กำหนด - property คือตัวระบุคุณสมบัติที่อยากเปลี่ยนหรือกำหนดค่าให้
value คือค่าที่กำหนดให้แก่ property โดยต้องมี colon (:) คั่นระหว่างชื่อ property กับค่าที่กำหนดให้ value
property และ value ต้องใส่ไว้ในเครื่องหมายวงเล็บปีกกา
property selector {property: "word word"} .class1.class2 child selector (>)
value selector {property: value; value; value;} .class1 .class2 adjacent sibling selector (+)
- selector{
property: value;
property: value;
property: value;
}		 #id general sibling selector (~)
- selector, selector, selector{
property: value;
property: value;
property: value;
}		 * -
