Property Value Unit Differentiation
คุณสมบัติ ค่าที่กำหนด - property คือตัวระบุคุณสมบัติที่อยากเปลี่ยนหรือกำหนดค่าให้
value คือค่าที่กำหนดให้แก่ property โดยต้องมี colon (:) คั่นระหว่างชื่อ property กับค่าที่กำหนดให้ value
property และ value ต้องใส่ไว้ในเครื่องหมายวงเล็บปีกกา
Property ที่สอนใน Pre-Camp มี 5 กลุ่ม
- Property for content: color , size , font , text, list-style-type, Height, Width, background
- Property for border : border, table
- Property for padding : padding (top right buttom left)
- Property for margin : margin (top right buttom left)
- Property for element : display, Height, Width, Position, Float, Align, Flex, Grid		 - - -
- Value unit มี 2 ประเภท คือ
- หน่วยแบบ Absolute Length (กำหนดตายตัว)
- หน่วยแบบ Relative Length (กำหนดแบบอัตราส่วน)		 - cm เซนติเมตร, mm มิลลิเมตร, in นิ้ว, px พิกเซล 1/96 นิ้ว, pt พอยท์ 1/72 นิ้ว, pc พิคาล 1/12 นิ้ว
- em เท่าขนาดฟอนต์ของ element, ex เท่าความสูงของฟอนต์ของ element, ch ความกว้างของตัวหนังสือใน parent element, rem เท่าขนาดฟอนต์ของ root element, vh เปอร์เซ็นต์ความสูงหน้าจอ, vw เปอร์เซ็นต์ความกว้างหน้าจอ, vmin เปอร์เซ็นต์ของ view point ต่ำสุด, vmax เปอร์เซ็นต์ของ view point สูงสุด, % เปอร์เซ็นต์ของ parent element		 -
